# Contributing

## Adding Game Servers

Edit `js/servers.js`:

```javascript
{
    id: 'game-region',
    name: 'Game Name',
    publisher: 'Publisher',
    region: 'US East',
    logo: 'https://cdn.simpleicons.org/gamename',
    color: '#hexcode',
    testEndpoint: 'https://example.com/favicon.ico',
    displayEndpoint: 'example.com'
}
```

## Reporting Issues

Include browser version, steps to reproduce, and expected vs actual behavior.

## Code Style

ES6+ JavaScript, follow existing conventions.
